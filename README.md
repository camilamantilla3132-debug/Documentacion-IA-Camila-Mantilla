# Primer Prompt 💪
- **Tema:** Musculos trabajados y Calorias quemadas en los diferentes estilos de la natacion.

- **Rol:** Entrenadora profesional de natación,ejercicio y preparación física.
- **Tarea:** Explicar qué músculos trabajan más en cada estilo, comparar los estilos y calcular las calorías quemadas según la distancia y la intensidad.
- **Contexto:** La información debe ser clara para estudiantes y deportistas intermedio,pero suficientemente detallada para comprender cómo afecta cada estilo al cuerpo.
- **Formato:** Para la comparacion haz una tabla comparativa en md.Y para cada estilo haz un enunciado.


```text
Actúa como una entrenadora profesional de natación,ejercicio y preparacion fisica.Explícame qué músculos y partes del cuerpo trabajan más al nadar libre, pecho y espalda,donde tienes que mostrar cuáles se fortalecen y cuáles se fatigan más. También estima las calorías que se queman al nadar distintas distancias con una intensidad moderada-alta que es la intensidad en la que yo nado. Organiza la información con tablas, subtítulos,enunciados y un resumen final para que sea fácil de entender.
```

# Quinto Prompt 💲
